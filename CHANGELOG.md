# Changelog

All notable changes to the LawnFlow.ai Marketing Website will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project setup and documentation

## [0.7.0] - 2026-01-18

### Added
- Clean black iPhone shell component for all mobile mockups
- Mobile app preview component with Owner, Crew, and Customer variants
- Animated mobile experience showcase carousel on home page
- iPhone mockups integrated across all pages
- Owner dashboard showcase section on For Owners page
- Crew mobile screens with route view, job checklist, and photo upload
- Customer mobile screens with dashboard, service history, and billing

### Changed
- Updated all iPhone mockups to use clean black frames (removed colored edges)
- Enhanced hero section with Owner mobile app preview
- Improved mobile experience presentation throughout site

### Technical
- Added Motion (Framer Motion) for smooth carousel animations
- Created reusable IPhoneShell component
- Created MobileAppPreview component with three variants
- Updated Screens page with new mobile screen components

## [0.6.0] - 2026-01-15

### Added
- Compare page with comprehensive competitive value matrix
- Seven accordion categories comparing major competitors
- Detailed feature comparison with Yardbook, Jobber, Housecall Pro, Service Autopilot, and Aspire
- Updated home page hero with "AI for Lawn and Landscaping, finally" messaging
- New "Get your nights back" value proposition
- Waitlist capture form in hero section
- Video demo section placeholder

### Changed
- Replaced old Competitive page with new Compare page
- Enhanced hero messaging and layout
- Improved CTA placement and prominence

## [0.5.0] - 2026-01-12

### Added
- Three Audiences Section on home page
- Elevated "For Crews" and "For Customers" to first-class status
- Updated header navigation to include all three audience types
- Enhanced messaging emphasizing AI orchestration for all three user types

### Changed
- Reorganized home page sections for better flow
- Updated navigation structure across all pages
- Improved mobile responsiveness for audience sections

## [0.4.0] - 2026-01-10

### Added
- Agents page with detailed AI agent information
- InvoiceBuilAgent, BillingAgent, RemediationAgent, and other agents
- Agent collaboration diagram
- Human-in-the-loop section
- Screens & Flows page with desktop and mobile previews
- Command Center, Quote Copilot, Schedule Optimizer screens
- User flows section with detailed workflows

### Changed
- Enhanced agent showcase with real system architecture
- Improved visual consistency across agent cards

## [0.3.0] - 2026-01-08

### Added
- For Owners page with dashboard showcase and before/after comparison
- For Crews page with mobile app preview and feature list
- For Customers page with customer portal and benefits
- Dedicated hero sections for each audience

### Changed
- Improved page layouts for better readability
- Enhanced mobile responsiveness across all audience pages

## [0.2.0] - 2026-01-05

### Added
- How It Works page with agent cards and lifecycle timeline
- Trust ladder section with progressive confidence building
- Waitlist page with email capture form
- Waitlist confirmation state and success messaging

### Changed
- Enhanced home page with problem/solution sections
- Improved capabilities and outcomes sections
- Added competitive callout section

## [0.1.0] - 2026-01-01

### Added
- Initial project setup with React 18, TypeScript, Vite 6, Tailwind CSS 4
- Comprehensive design system with brand colors, typography, and spacing
- Home page with hero, problem, solution, and capabilities sections
- Header and Footer components with full navigation
- Reusable UI component library (50+ components)
- Responsive breakpoints for mobile, tablet, and desktop
- React Router setup with client-side routing

### Technical
- Vite build configuration with path aliases
- Tailwind CSS v4 with custom theme
- CSS custom properties for design tokens
- Radix UI component primitives
- Lucide React icon library
